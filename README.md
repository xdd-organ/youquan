# youquan
优荃网站

```
    1.安装   npm install
    2.运行   npm run dev
    3.编译   npm run build
```

```
    1.添加百度地图  npm i --save vue-baidu-map
        百度地图使用：<baidu-map></baidu-map>
    2.添加Swiper（图片滑动）   npm install swiper --save
        main.js添加    import 'swiper/dist/css/swiper.css';
        需要用的组件添加 import Swiper from 'swiper';
```
